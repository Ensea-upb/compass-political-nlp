# Taxonomy

The public taxonomy is intentionally compact:

| Theme | Scope |
| --- | --- |
| economy | Jobs, investment, industry, taxation, inflation |
| sovereignty | National independence, borders, self-determination |
| democracy | Elections, transparency, institutions, rights |
| social_policy | Schools, health, housing, youth |
| environment | Climate, energy, water, land |

This compact layer is a placeholder for richer manifesto, V-Party, CMP, or project-specific coding schemes.
