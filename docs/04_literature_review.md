# Literature Review

The public repository does not redistribute downloaded articles or private PDFs.

The literature layer should summarize concepts rather than store copyrighted material:

- manifesto coding and comparative party-position analysis;
- retrieval-augmented evidence selection;
- multilingual political text classification;
- validation and uncertainty reporting in computational social science.

Full bibliographic references can be added as manually curated citations.
